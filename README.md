# Sigi-MLIR

MLIR middle-end for the Sigi compiler. Part of the Komplexpraktikum MLIR.

See also [the frontend](https://github.com/oowekyala/sigi-frontend/).



## Before building

Register local LLVM and MLIR via 

``` shell
export MLIR_DIR=~/path/to/kp-mlir/llvm-project/build-Debug/lib/cmake/mlir
export LLVM_INSTALL_DIR="~/path/to/kp-mlir/llvm-project"
```
